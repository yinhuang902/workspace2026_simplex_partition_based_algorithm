# Test Problem Catalog

All problems are two-stage stochastic programs of the form:
```
min_x  E_ω[ Q(x, ω) ]
s.t.   x ∈ X
```
where `x` is the first-stage decision, `ω` indexes scenarios, and `Q(x, ω)` is the recourse (second-stage) objective evaluated by fixing `x` and solving the full model under scenario `ω`.

Stochastic scenarios are generated by perturbing constraint RHS values using Julia-compatible RNG (MT19937, seed=1234).

---

## 2_1_1 — Nonconvex QP (5 variables)

**Type:** Nonconvex quadratic objective, single linear constraint

| Property | Value |
|----------|-------|
| Variables | x1..x5 ∈ [0, 1] |
| First-stage | x1, x2 (dim=2) |
| Scenarios | 1000 (Plasmo.jl) |

**Formulation:**
```
min   42*x1 + 44*x2 + 45*x3 + 47*x4 + 47.5*x5
      - 50*(x1² + x2² + x3² + x4² + x5²)

s.t.  20*x1 + 12*x2 + 11*x3 + 7*x4 + 4*x5 <= b    (b = 40, stochastic)
      0 <= xi <= 1,  i = 1..5
```
**Stochastic:** `b_ω = 40 * U(0.5, 2.0)`

## 2_1_2 — Nonconvex QP (6 variables)

**Type:** Nonconvex quadratic objective, two linear constraints

| Property | Value |
|----------|-------|
| Variables | x1..x5 ∈ [0, 1], x6 ≥ 0 |
| First-stage | x1, x2 (dim=2) |
| Scenarios | 1000 (Plasmo.jl) |

**Formulation:**
```
min   -0.5*(x1² + x2² + x3² + x4² + x5²)
      - 10.5*x1 - 7.5*x2 - 3.5*x3 - 2.5*x4 - 1.5*x5 - 10*x6

s.t.  6*x1 + 3*x2 + 3*x3 + 2*x4 + x5 <= b         (b = 6.5, stochastic)
      10*x1 + 10*x3 + x6 <= 20
      0 <= xi <= 1 (i=1..5),  x6 >= 0
```
**Stochastic:** `b_ω = 6.5 * U(0.5, 2.0)`

## 2_1_3 — Nonconvex QP (13 variables)

**Type:** Nonconvex quadratic objective, nine linear constraints, block structure

| Property | Value |
|----------|-------|
| Variables | x1..x9 ∈ [0,1], x10..x12 ≥ 0, x13 ∈ [0,1] |
| First-stage | x1, x2 (dim=2) |
| Scenarios | 1000 (Plasmo.jl) |

**Formulation:**
```
min   5*(x1 + x2 + x3 + x4) - 5*(x1² + x2² + x3² + x4²)
      - (x5 + x6 + x7 + x8 + x9 + x10 + x11 + x12 + x13)

s.t.  2*x1 + 2*x2 + x10 + x11 <= b                 (b = 10, stochastic)
      2*x1 + 2*x3 + x10 + x12 <= 10
      2*x2 + 2*x3 + x11 + x12 <= 10
      -8*x1 + x10 <= 0
      -8*x2 + x11 <= 0
      -8*x3 + x12 <= 0
      -2*x4 - x5 + x10 <= 0
      -2*x6 - x7 + x11 <= 0
      -2*x8 - x9 + x12 <= 0
```
**Stochastic:** `b_ω = 10 * U(0.5, 2.0)`

## 2_1_7 — Nonconvex QP (20 variables, circulant constraints)

**Type:** Nonconvex quadratic objective, 10 linear constraints (circulant-like pattern)

| Property | Value |
|----------|-------|
| Variables | x1..x20 ≥ 0 |
| First-stage | x1..x5 (dim=5) |
| Scenarios | 100 (PlasmoOld) |

**Formulation:**
```
min   -0.5 * Σ_{i=1}^{20} i * (xi - 2)²

s.t.  c1..c5: sparse linear ≤ constraints        (c1-c5 RHS stochastic)
      c6..c9: sparse linear ≤ constraints         (deterministic)
      c10: x1 + x2 + ... + x20 <= 40
      xi >= 0,  i = 1..20
```
**Stochastic (PlasmoOld):** `c_k_rhs_ω = c_k_rhs + |c_k_rhs| * U(0, 2)` for c1-c5.

First-stage bounds: x1..x5 ∈ [0, 8] (practical limit; implied bound from c10 is 40).

## 2_1_8 — Nonconvex QP (24 variables, equality constraints)

**Type:** Nonconvex quadratic objective, 10 constraints (6 equalities + 4 equalities)

| Property | Value |
|----------|-------|
| Variables | x1..x24 ∈ [0, 100] |
| First-stage | x1..x5 (dim=5) |
| Scenarios | 100 (PlasmoOld) |

**Formulation:**
```
min   Σ_{i=1}^{24} (L_i * xi - Q_i * xi²)

      where L = [300, 270, 460, 800, 740, 600, 540, 380, 300, 490,
                 380, 760, 430, 250, 390, 600, 210, 830, 470, 680, 360, 290, 400, 310]
            Q = [7, 4, 6, 8, 12, 9, 14, 7, 13, 12,
                 8, 4, 7, 9, 16, 8, 4, 10, 21, 13, 17, 9, 8, 4]

s.t.  x1 + x2 + x3 + x4 == 8                     (first-stage only, NOT modified)
      x5 + x6 + x7 + x8 == 24                     (stochastic: split into ≤/≥)
      x9 + x10 + x11 + x12 == 20                  (stochastic)
      x13 + x14 + x15 + x16 == 24                 (stochastic)
      x17 + x18 + x19 + x20 == 16                 (stochastic)
      x21 + x22 + x23 + x24 == 12                 (stochastic)
      x1 + x5 + x9 + x13 + x17 + x21 == 29       (deterministic)
      x2 + x6 + x10 + x14 + x18 + x22 == 41      (deterministic)
      x3 + x7 + x11 + x15 + x19 + x23 == 13      (deterministic)
      x4 + x8 + x12 + x16 + x20 + x24 == 21      (deterministic)
      0 <= xi <= 100,  i = 1..24
```
**Stochastic (PlasmoOld):** Each stochastic equality is split: UB via `a + |a|*U(0,2)`, LB via `a + |a|*U(-2,0)`.

**Note:** c1 (first-stage equality) makes the box-corner simplex initialization problematic.  First-stage bounds tightened to x1-x4 ∈ [0,8], x5 ∈ [0,24].

## 2_1_10 — DC Quadratic (20 variables)

**Type:** Difference-of-convex (DC) quadratic objective, 10 linear constraints

| Property | Value |
|----------|-------|
| Variables | x1..x20 ≥ 0 |
| First-stage | x1..x5 (SNGO) or x1..x2 (Plasmo.jl) |
| Scenarios | 100 (SNGO) or 1000 (Plasmo.jl) |

Two runner variants exist:
- `run_2_1_10_case.py` — Plasmo.jl version (nfirst=2, nparam=2)
- `run_2_1_10_sngo_case.py` — PlasmoOld/SNGO version (nfirst=5, nparam=5)

**Formulation:**
```
min   0.5 * pos(x) - 0.5 * neg(x)

pos = 42*(52+x11)² + 98*(3+x12)² + 48*(x13-81)² + 91*(x14-30)²
    + 11*(85+x15)² + 63*(x16-68)² + 61*(x17-27)² + 61*(81+x18)²
    + 38*(x19-97)² + 26*(73+x20)²

neg = 63*(19+x1)² + 15*(27+x2)² + 44*(23+x3)² + 91*(53+x4)²
    + 45*(42+x5)² + 50*(x6-26)² + 89*(33+x7)² + 58*(23+x8)²
    + 86*(x9-41)² + 82*(x10-19)²

s.t.  c1..c9: dense linear ≤ constraints (c1-c5 stochastic)
      c10: x1 + x2 + ... + x20 <= 200
      xi >= 0,  i = 1..20
```

## 14_1_6 — Nonconvex QCQP (9 variables)

**Type:** Linear objective, bilinear + quadratic (sphere) constraints

| Property | Value |
|----------|-------|
| Variables | x1..x8 ∈ [-1, 1], x9 free |
| First-stage | x1, x2 (dim=2) |
| Scenarios | 1000 (Plasmo.jl) |

**Formulation:**
```
min   x9

s.t.  0.004731*x1*x3 - 0.1238*x1 - 0.3578*x2*x3 - 0.001637*x2
      - 0.9338*x4 + x7 - x9 <= b                  (b = 0.3571, stochastic)

      [c2..c6: bilinear inequality constraints]
      -0.7623*x1 + 0.2238*x2 == -0.3461            (linear equality)

      x1² + x2² == 1                               (sphere, via ≤ and ≥)
      x3² + x4² == 1
      x5² + x6² == 1
      x7² + x8² == 1

      -1 <= xi <= 1  (i=1..8),   x9 free
```
**Stochastic:** `b_ω = 0.3571 * U(0.5, 2.0)`

**Note:** This is the most structurally different problem — nonconvex QCQP with bilinear terms in constraints and unit-sphere constraints. Requires `NonConvex=2` for Gurobi.

## ST_FP7a — Nonconvex QP (20 variables, variant of 2_1_7)

**Type:** Nonconvex quadratic objective, same constraint structure as 2_1_7

| Property | Value |
|----------|-------|
| Variables | x1..x20 ≥ 0 (x1-x5 individually bounded) |
| First-stage | x1..x5 (dim=5) |
| Scenarios | 100 (PlasmoOld) |

**Formulation:**
```
min   Σ (2*xi - 0.5*xi²)   for the 20 variables

s.t.  Same circulant constraint structure as 2_1_7
      c1-c9 stochastic, c10: sum(xi) <= b
```

## ST_RV2 — Nonconvex QP (20 variables, random dense)

**Type:** Nonconvex quadratic objective, dense linear constraints

| Property | Value |
|----------|-------|
| Variables | x1..x2 ∈ [0, 200], x3..x20 ≥ 0 |
| First-stage | x1, x2 (dim=2) |
| Scenarios | varies |

**Formulation:**
```
min   Σ_{i=1}^{20} (-a_i * xi² - b_i * xi)

s.t.  c1-c10: dense linear ≤ constraints (c1-c5 stochastic)
```
